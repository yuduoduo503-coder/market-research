## 视觉规范

所有报告 HTML 遵循「元任务全场景视觉规范 Design System v1.1」：
- 深夜蓝主色 `#0F172A` + 蓝紫渐变强调 `#38BDF8 → #818CF8`
- Slate 灰阶 + 8pt 栅格 + 圆角方块气质
- Inter + Noto Sans SC + JetBrains Mono 字体体系
- Logo 本体以原 VI 为准，不可重绘改色

## 更新方式

新增调研报告时：
1. 在 `reports/` 下按主题新建目录
2. 放入 HTML 与对应 PDF
3. 更新本 README 的「已归档报告」表格
4. 提交并推送，GitHub Pages 自动更新
